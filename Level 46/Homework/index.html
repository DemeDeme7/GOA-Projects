CSS-ის საზომი ერთეულები

ერთეული აღწერა მაგალითი
px პიქსელი width: 200px;
% პროცენტული მნიშვნელობა მშობელი ელემენტის ზომის მიმართ width: 50%;
em ელემენტის ფონტის ზომის მიმართ font-size: 2em; /* ორჯერ დიდი */
rem ფონტის ძირითადი ზომის მიმართ font-size: 1.5rem; /* 1.5 * root font size */
vh ეკრანის სიმაღლის პროცენტული მნიშვნელობა height: 50vh; /* ეკრანის 50% სიმაღლე */
vw ეკრანის სიგანის პროცენტული მნიშვნელობა width: 80vw; /* ეკრანის 80% სიგანე */
vmin ეკრანის პატარა მხარის 1% width: 10vmin;
vmax ეკრანის დიდი მხარის 1% width: 10vmax;
