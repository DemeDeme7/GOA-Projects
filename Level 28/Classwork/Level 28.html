Inline CSS-ი გამოიყენება CSS-ის კოდის თვით თეგში (კოდის ხაზში) დასაწერად. Internal CSS-ი გამოიყენება თეგში თეგით CSS-ის კოდის დასაწერად. External CSS-ი გამოიყენება CSS-ის კოდის დასაწერად ცალკე(სხვა CSS) ფაილში. დეკლარაცია არის კოდის გამოძახება და სტრუქტურა,მაგალითად: color: red; აქ color არის თვისება,ხოლო red არის მისის მნიშვნელობა. id-ს ვანიჭებს თვით ელემენტს,გამხსნელ თეგში,როგორც დამატებით ატრიბუტად.(მაგ:id="#") Class სექტორი არის CSS-ის ელემენტი,რომელიც იზიარებენ ერთნაირ მოცულობას.(class="#")P.S. იწერება ისევე, როგორც id. element sector-ი არის CSS ელემენტი,რომელიც ცვლის იმ თეგის მოცულობის ყველა ელემენტს.
