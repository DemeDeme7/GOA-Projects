Windows XP


Windows XP is a major release of Microsoft's Windows NT operating system. It was released to manufacturing on August 24, 2001, and later to retail on October 25, 2001. It is a direct successor to Windows 2000 for high-end and business users and Windows Me for home users.

Windows XP shipped with 3 main editions.The chart shows what each edition includes and their key features:

Edition Key Features
Windows XP Home Edition - Designed for home users and personal use.
- Basic user interface with limited networking support.
- Single user and no Active Directory support.
- No Remote Desktop.
- Basic networking (no domain support).
Windows XP Professional - Designed for business and power users.
- Supports domain join and Active Directory.
- Includes Remote Desktop.
- Higher security features (Group Policy, Encrypting File System).
- Networking support for file and printer sharing.
Windows XP Media Center Edition - Special edition for home entertainment systems.
- Includes Windows Media Player and Media Center for TV, movies, music, and photos.
- Supports TV tuner cards.
- Can be used with remote control.
